* image thumbnail
When run ~image-dired~ under a directory of pictures, it will show thumbnails
for preview.

Press ~RET~ on a thumbnail will invoke ~image-dired-external-viewer~ to open the
image.

Package ~image+~ adds more additional image functionality.

* auto-save files
~#~ will flag all auto-save files (files whose names start and end with ~#~) for
deletion

* backup files
=~= will flag all backup files (files whose names end with =~=) for deletion
